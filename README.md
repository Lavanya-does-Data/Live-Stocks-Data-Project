# Live-Stocks-Data-Project
Description:
This project fetches **live stock prices** for multiple companies (e.g., AAPL, MSFT, GOOGL, AMZN, TSLA) using Python and the `yfinance` library. The data is processed with Pandas to calculate daily % changes, rolling averages, and simple buy/sell signals. Visualizations are created with Plotly.

Features:
- Fetch live stock prices
- Calculate daily % change and 7-day rolling average
- Generate simple buy/sell signals based on trend
- Visualize stock price trends interactively

Technologies:
- Python
- Pandas
- Plotly
- yfinance

How to Run: Install required libraries: pip install yfinance pandas plotly
